# Проект для сдачи 12-недельной аттестации
## Веб-приложение **"Планировщик задач"**
### Веб-приложение для управления задачами на ASP.NET и React.

Технологии:

1.Backend: ASP.NET Core, PostgreSQL, Entity Framework

2.rontend: React, Chakra UI, Tailwind CSS

Возможности:
+ Создание, редактирование, удаление задач
+ Установка сроков выполнения
+ Поиск и фильтрация задач
+ Отметка выполнения задач
+ Визуальные предупреждения о сроках
+ GET/POST /notes - получение и создание задач
+ PUT/DELETE /notes/{id} - обновление и удаление
+ PATCH /notes/{id}/toggle-complete - отметка выполнения
